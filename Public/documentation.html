My Favorite Films

This App will allow users to find information about their favorite films. Users will have the ability to create a profile with lists of their favorite movies.

Action Request URL Response
Return Every Movie GET '/movies' Returns an array of all JSON movie objects within
Return a single Movie GET '/movies/:title' Returns a single JSON movie title object specified in the url
Returns A Director GET '/movies/:director' Returns a single JSON director object specified in the url
Returns A Genre GET '/movies/:genre' Returns a single JSON genre object specified in the url
Allow Users to create an account POST '/users' Returns the JSON Object that that the user passed into the request
Allow Users to update their account information PUT '/users/:Username' Returns the JSON Object that that the user updated
Allow Users to delete their account information DELETE '/users/deleteAccount' Returns the a text confirmation that the account was deleted
